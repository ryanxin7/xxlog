## 代词



```
1.few 和 little、some和any (any多用否定句)、many和 much 是冤家，both、either 经常
考

1.few、little 稀少，极少 = 没有
(1)few 作主语时，谓语动词用复数，多用于否定句。
(2)little 作主语时，谓语动词用单数，多用于否定句。

2. a few、a little 一些，少量
(1)a few 作主语时，谓语动词用复数，多用于肯定句。
(2)a little 作主语时，谓语动词用单数，多用于肯定。

few + 可数  little + 不可数
```



```
3.any一些

any 多用于否定句、疑问句和条件状语从句中。
当句中含有“任何”的意思时，any可用于肯定句。
例句: Here are three novels. You may read any.这有三本小说，你可以任读一本。
```



```
4.many 、much 许多

mary+可数名词，much+不可数名词。
例句:How many people are there at the meeting?  有多少人参加会议?
How much time has we left? 我们还剩多少时间?
Many of the workers were at the meeting. 许多工人都参加了会议。
```



```
5.one/the other/anther/others/the others/the rest 固定搭配要分清

1.one/another/the other 用法
one... the other...一个...另一个(只有两个) 剩下的一个特指
one...another..一个...另一个(三个及以上) 剩下的两个中的随便一个
some... the others...有三个以上
one...another (one) ..., the other (a third)

the rest  = the others 剩余的全部

(1)泛指另一个用another。
(2)一定范围内的两个人(物)，一个用one，另一个用 the other。
(3)一定范围内的三个人(物)，一个用 one，另一个用one(another)，第三个可用the other 或a third。
(4)一定范围内，除去一部分人/物，剩余的全部用 the others 。
(5)泛指别的人或物时，用others;当在一定范围内，除去一部分后的剩余部分(但不是全部)，也用others。
```



```
6.代词常考固定搭配，阅读理解常考察: 
every other 每隔一个;none/no other than 不是别人正是;anything but 根本不，绝不;nothing but 只有,只不过 rather than 而不是;

例:I went home every other week.我每隔一周回一次家。
The man was none other than my husband.那个人(不是别人)正是我的丈夫。
The hotel was anything but cheap.这家旅馆根本不便宜。
Daisy is nothing but a teacher.黛西只不过是个老师。
What made you choose the one rather than the other?
```





## 三、比较级

```
1.比较级和最高级的规则不常考，结构形式要牢记:

(1)两者考是比较级，结构形式:more/形容词或副词 er+than 。
其中:形容词或副词是单音节和少数双音节(两个元音在一起)时，形容词或副词直接加er，多音节时，形容词或副词前用 more;

表示两者的比较时，通常用句型“比较级+than" 的结构 
He gets to school earlier than the other students.他比其他学生上学早。
My major is more difficult than his.  我的专业比他的专业更难。

(2)三者及以上用最高级。结构形式:the+most/形容词或副词+est 。
其中:单音节和少数双音节（两个元音在一起)时，形容词或副词后直接加est，多音节时，形容词或副词前 most(副词最高级前 the 可以去掉)

表示三者及以上的比较时，通常用句型“the(副词可省略)+most/~est 的结构
She is the most active student in our class.她是我们班最积极的学生。
She chose the most expensive of the hotels. 她选择了最贵的旅店。
Lily runs fastest in the class 李莉是全班跑得最快的。
```





```
2.比较级两个固定搭配:
(1)the more(或比较级)...,the more(或比较级) ...越...,越
(2)比较级+and+比较级 越来越...

The more you practice English, the better you will learn it. 你做得练习越多，英语学得越好。
The days are getting longer and longer. 天越来越长了。
```



```
3.拉丁语的形容词(or为结尾)后面要接to;

inferiorto 比.....·差;superior to 比.....好;prior to 在......之前; junior to 比......年少 senior to 比......年长

例题:
1.I got the news ____ your call, but thank you just the same.
A. due to B. used to C. senior to D. prior to 
答案: D
句意:你给我打电话之前我就得知了这个消息，但还是很感谢你。

2.I hate those people who think they are superior _____ me without any reason.
A. a B. on C. to D.than

补充： be used to do 、used to doing 、used to do 区别
 be used to do 被用来做
 used to doing 习惯于做...
 used to do 过去常常做...
```



```
4.同级比较: 表示两者是同等程度或不同程度时，通常用句型"as 原级 +as...” 或“  not as (so)+ 原级+as...”

例题:
There are not ____ students in Class One as in Class Two.
A. so many B. so much C. more D. much more
答案:A句意:一班的学生没有二班的学生多。
```



```
5.比较级前加修饰词一定要小心，首选 much 还有 far

其他修饰比较级的词: even、still、a lot、a bit、a little rather、 slightly 和 by far 等。
例句: He gets to school much earlier than the other students.
```



```
6.倍数表示法经常考:
倍数 times+形容词比较级+than 倍数 
    times as+形容词原级+as 倍数 表示法最常考:
    twice as much/many(注意可数与不可数)+as
例句:
The dining hall is three times larger than that one.
这个餐厅是那个餐厅的大三倍。
The dining hall is three times as large as that one.
这个餐厅是那个餐厅的三倍大。

例题:
The fried fish we ate at the restaurant yesterday isdelicious. 
I'd like to have it again even if it costs____.
A. as twice much C. much as twice B. twice as much D. as much twice
答案:B
句意:我们昨天在饭店吃的炸鱼很好吃。尽管现在价格是原来的两倍，我还是想再吃。
```


## 五、连词

```
1.连词中文要记牢，并列、转折和原因，条件、方式和目的，这六类连词是必考

(一)、一定要记住以下连词的中文释意
1.and: 和，与;而且;于是，然后，因此。
2.but:但是;常用 not...but...,表示不是....··而是..... ;
3.or :或者，用于否定句或疑问句中表示“也不”，否则，要不然;也就是说，换言之。
例:
There is no air and no water on the moon.
或者 There is no air or water on the moon.

4.nor: 用在 neither 之后表示“也不”;
5.so 因此，所以;因而，从而。
6.yet: 可是，却，然而。
7.for: 因为（表示推测)，由于。
8.both...and:既.....又...... ;两个都。
9. not only...but also : 不但.. ...而且
10.either..or: 不是....·就是，要么.....要么。
11.neither...nor: 既不......也不。
12. A as well as B:不但 A 而且 B。

(二)、一定要记住以下连接副词的中文释义
1. besides、furthermore、moreover、in addition 都有“而且”的意思。
2.yet、still、however、nevertheless 都有“但是”的意思。
3.else和 otherwise都有“否则”的意思。
4. thus、hence、therefore 、 accordingly 、 consequently都有“因此，所以”的意思。

(三)引导状语从句的连词
1.表示时间: when、while (“当...的时候”，从句要用进行时，“尽管”“然而”)、as、whenever 无论何时;since、 as soon as 一旦。
2.表示条件:if 如果、unless 除非; as longas 只要;incase万一，如果;if only 只要。
3.表示结果: so...that、 such...that 如此.....·以至于
4.表示目的: so that 以便; in order that 为了，以便;in case 以防，以备。
5.表示原因: as, because, now(that),since...由于。
6.表示让步: whether..or 是否;though 虽然;although 尽管;while 虽然。
7.表示地点: where、wherever 无论什么地方。
8.表示方式: as,as...so,asif 好像,仿佛,似乎;according to 根据。
```

## 六、时态

```
1.一般现在时

1.构成:动词用原形、第三人称单数形式。
2.用法:
(1)表示经常发生的动作、习惯性的动作或存在的状态，常与时间状语连用。
时间词: usually、always、sometimes、 often 、 every day
(2)表示永恒的客观规律或真理，即使出现在过去的语境中，仍用一般现在时。

(3)主将从现:
例题.if it ______ tomorrow, we won’t go for a picnic.
A. Will rain B. should rain C. rains D. rained 答案:C
句意:如果明天下雨，我们就不去野餐了。解析:本题考查主将从现。主句是一般将来时，从向应该是一般现在时。
故选 C。
```



```
2.一般般过去时
1.构成:由动词过去式构成。动词 be的过去式根据人称有was 和were 两种变化。规则动词变过去式在词尾加d或ed。
2.用法:
(1)表示在过去某一时间内发生的动作或存在的状态常与表示过去的时间状语连用。
如 yesterday、the otherday、justnow、ago、last week 等，以及 when 等连词引导的状语从句。
有时句子中的时间标志不是很明显，但通过上下文可以看出是过去的动作或状态。

(2)表示在过去经常发生的动作。
```



```
3.现在完成时
1.构成:由助动词 have、has 加过去分词构成。
2.用法:
(1)表示过去发生的某一动作对现在造成的影响或结果。
常与 never、 already、 not...yet 、 just、by this time等状语连用。
(2)“for+ 时间段”或“since+ 语(或从)”的状语连用。
```



```
4.过去完成时
1.构成:由助动词had 加动词的过去分词构成
2.用法:
(1)表示在过去某一时刻或动作之前已经完成的动作即“过去的过去”。
常与after、before、by、until、once、when 等介词或连接词引出的表示到过去某一时间为止的时间状语连用。
(2)表示由过去某一时间开始，一直延续到过去另一时间的动作或状态。常与for 或since 构成的时间状语连用。


```



```
5.将来完成时
1.构成:由助动词 shall、will +have+ 过去分词构成。
2.用法:表示在将来某一时刻之前或在另一个未来的动作发生之前已经完成的动作。
常用的时间状语一般用 by加将来的时间表示。
如:by the end of this year、by 8oclock,以及由 bythetime...、before 或 when 等引导的时间状语从句。
```





## 七、语态

```
1.ing 表被动的动词要记清 want/need/require+doing 相当于want/need/requiret to be done

例句: The house needs repairing.————> The house needs to be repaired.
例题 : The flowers in the garden need ____ badly , otherwise they will dry up soon.
A. watering B. to water C. watered D. to have watered
答案:A句意:花园里的花急需浇水。否则它们很快就会干枯。
```



```
2.被动语态记结构一般连着时考，时态变换 be 动词，真实主语在by 后;
形式为 be+及物动词过去分词,be 根据主语的人称、数时态和语气的变化而变化。
例:
1. The flowers is watered by an old man every day.有个老人每天都给花浇水。(一般现在时的被动语态)
2. The flowers used to be watered by an old man.有个老人过去常常给花浇水。(一般过去时的被动语态)
3. A new station was being built near the zoo at this timelastyear.去年这个时候，这个车站附近正在修建一座车站。(过去进行时的被动态)
4. English will have been studied for a year by the end ofnextyear.截至明年年底，我们已经学习英语一年了。(将来完成时的被动语态)
```





## 八、情态动词

```
1.情态动词考基础用法要牢记
can/could 表示“可能，能够”
may/might 表示“很有可能”
must 语气最强，“一定，必须”
shall/should、ought to 表示“应该”，理论上的可能性

must 90%
might 50%
```



```
2.must 和 have to 的用法
must 表示“必须”，用于肯定句或疑问句。
其否定式mustn't 表示“禁止”，疑问句的否定回答常用needn't或don't have to 表示“不必”。
have to与其他情态动词一样也是后接动词原形表示外界的强制作用，强调被迫性，含有不情愿的色彩,常译为“必须,不得不”。
```



```
3.need 的用法表示“需要”或“必须”，作情态动词时，仅用于否定句或疑问句中 neend't 表示“不必”。
在肯定句中一般用must、have to 、ought to 或 should 代替。
```



```
4.should、ought to 和 had better 的用法 should、ought to 和 had better 均为提建议型情态动词而含义及语气有所不同。
had better+do 语气最弱，意为“最好”;
should 语气较强，意为“应该”;
ought to语气最强，意为“应该”。should 与ought to 常可通用
```



```
5.情态动词表推测
1.肯定性推测用情态动词must。
对过去事物的推测用 must have done,其中 must 表示肯定的推测，意为“一定”。

2.对可能性推测:
表示对发生过的事情的推测 might have done(可能性没有 must大)
3.否定性推测用情态动词 can't;对过去事物的推测用 can'thavedone.其中，can't/couldnt 表示否定的推测，意为“不可能”。


(1) The teacher_____overtime last night, for his lightwas on all the time.
A. must have workedB. should have done C. could have done D. must work
答案选 A句意:老师昨晚一定是加班了，因为他的灯一直亮着。
(2) He _____ you more help, even though he was busy
A. might have given B. might give D. could have given C. must give
答案:A句意:他或许会多给你一些帮助，即使他很忙。
(3)I think that he couldn't/can't have gone abroad. I saw him just now.我认为他不可能出国了。我刚还看见他了
```





## 九、主谓一致



```

together with 和......一起; along with 随同....一起跟·.....一起;

```

```

either...or.. 要么......要么...... neither...nor... 既不...... 也不... ... ;not only...but also... 不但......而且.....;not...but... 不是......而是...... ;or 或者;there be 句型

例题: Neither Bill nor his parents _____ at home.
A.is B. are C.has D. have
答案:B句意:比尔和他的父母都不在家。
```

